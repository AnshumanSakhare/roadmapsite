# GenAI Learning Roadmap

Welcome to the **Gen-AI-Experiments** learning curriculum. This roadmap guides you from complete beginner to advanced GenAI developer in 7 structured levels.

## 📊 Roadmap Overview

```
Level 0: Prerequisites ─────→ Setup & Foundations
    ↓
Level 1: LLM Basics ────────→ Understand Models & APIs
    ↓
Level 2: Simple GenAI Apps──→ Build Simple Applications
    ↓
Level 3: RAG + Agents ──────→ Advanced Patterns
    ↓
Level 4: Production ────────→ Deploy Real Systems
    ↓
Level 5: Advanced GenAI ────→ Cutting-Edge Techniques
    ↓
Level 6: Specialization ────→ Choose Your Path
```

## 🎓 How to Use This Roadmap

1. **Start at Level 0** - Don't skip prerequisites
2. **Follow the progression** - Each level builds on the previous one
3. **Complete checkpoint tasks** - Hands-on exercises in each level
4. **Run the projects** - Learning by building is most effective
5. **Join the community** - Share your progress and help others

---

## 💡 Learning Philosophy

- **Learn by Building** - Every concept taught with a working project
- **Bite-Sized Lessons** - 1-2 hour focused sessions
- **Practical Focus** - No theory without code
- **Production-Ready** - Build real applications, not just tutorials

---

## ⏱️ Total Time Commitment

- **Beginner to Level 3**: 10-15 weeks (part-time)
- **Beginner to Level 4**: 16-23 weeks (part-time)
- **Beginner to Level 5**: 24-35 weeks (part-time or full-time)
